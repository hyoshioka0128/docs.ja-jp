# <a name="breaking-changes-in-windows-forms-net-framework-to-net-core"></a>Windows フォームの破壊的変更 (.NET Framework から .NET Core)

WPF と Windows フォームのサポートが .NET Core にバージョン 3.0 で追加されました。 .NET Framework から .NET Core に Windows フォームまたは Windows Presentation Foundation のアプリを移行する場合、この記事の一覧にある破壊的変更がご使用のアプリに影響する可能性があります。

破壊的変更は、それが導入された .NET Core のバージョン別に分類されています。
