# <a name="compiler-warning-level-2-cs0162"></a>コンパイラの警告 (レベル 2) CS0162

到達できないコードが検出されました
  
実行されないコードが検出されました。

## <a name="example"></a>例

次の例では、CS0162 が生成されます。

```csharp
// CS0162.cs
// compile with: /W:2
public class Program
{
    public static void Main()
    {
        goto lab1;
        {
            // The following statements cannot be reached:
            int i = 9;   // CS0162
            i++;
        }
    lab1:
        {
        }
    }
}
```
