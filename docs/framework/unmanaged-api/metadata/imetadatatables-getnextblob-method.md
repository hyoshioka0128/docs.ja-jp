# <a name="imetadatatablesgetnextblob-method"></a>IMetaDataTables::GetNextBlob メソッド
テーブル内の次のバイナリラージオブジェクト (BLOB) のインデックスを取得します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT GetNextBlob (  
    [in]  ULONG   ixBlob,  
    [out] ULONG   *pNext  
);  
```  
  
## <a name="parameters"></a>パラメーター  
 `ixBlob`  
 からBlob の列から返されるインデックス。  
  
 `pNext`  
 入出力次の BLOB のインデックスを指すポインターです。
