# <a name="imetadatatablesgetnextstring-method"></a>IMetaDataTables::GetNextString メソッド
現在のテーブル列の次の文字列のインデックスを取得します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT GetNextString (   
    [in]  ULONG   ixString,  
    [out] ULONG   *pNext  
);  
```  
  
## <a name="parameters"></a>パラメーター  
 `ixString`  
 から文字列テーブル列からのインデックス値。  
  
 `pNext`  
 入出力列内の次の文字列のインデックスへのポインター。
