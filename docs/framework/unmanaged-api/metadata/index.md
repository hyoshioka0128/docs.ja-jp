# <a name="metadata-unmanaged-api-reference"></a>メタデータ (アンマネージ API リファレンス)
メタデータ API を使用すると、コンパイラなどのクライアントは、共通言語ランタイム (CLR: Common Language Runtime) によって読み込まれる型を使用せずに、コンポーネントのメタデータを生成またはアクセスできます。
