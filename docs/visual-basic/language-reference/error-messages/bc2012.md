# <a name="cant-open-filename-for-writing"></a>'\<filename > ' を書き込み用に開くことができません
指定されたファイルは既に開かれているため、書き込み用に開くことができません。  
  
 **エラー ID:** BC2012  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1. ファイルを閉じて、再度開きます。  
  
2. ファイルのアクセス許可を確認します。
