 プロシージャ呼び出しでは、イベント プロシージャ名を指定します。 イベント ハンドラーは、プロシージャが、イベント自体はシグナル デバイスで発生して処理する必要がありますが、です。  
  
 **エラー ID:** BC32022  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには
