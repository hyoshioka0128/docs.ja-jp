# <a name="labels-that-are-numbers-must-be-followed-by-colons"></a>数字を指定するラベルの後には、コロンが必要です。
行番号は、他の種類のラベルと同じ規則に従い、コロンが含まれている必要があります。  
  
 **エラー ID:** BC30801  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 数字の後にコード行の先頭にコロンを付けます。例えば：  
  
    ```vb  
    400:    X += 1  
    ```
