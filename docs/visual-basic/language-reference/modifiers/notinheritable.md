# <a name="notinheritable-visual-basic"></a>NotInheritable (Visual Basic)
クラスを基底クラスとして使用できないことを指定します。  
  
## <a name="remarks"></a>コメント  
 **代替語句**。 継承できないクラスは、*シール*クラスと呼ばれることもあります。  
  
 このコンテキストでは、`NotInheritable` 修飾子を使用できます。
