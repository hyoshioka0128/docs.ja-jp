# <a name="mycomputerinfo-object"></a>My.Computer.Info オブジェクト

 `My.Computer.Info` オブジェクトのプロパティの詳細については、「<xref:Microsoft.VisualBasic.Devices.ComputerInfo>」を参照してください。
