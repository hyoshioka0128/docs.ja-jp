# <a name="equality-operations-visual-basic"></a>等値演算 (Visual Basic)
2 つのシーケンスの対応する要素が等しく、要素数も同じである場合、これらのシーケンスは等しいと見なされます。

|メソッド名|説明|Visual Basic クエリ式の構文|詳細|
