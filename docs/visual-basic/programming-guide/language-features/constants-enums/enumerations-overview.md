# <a name="enumerations-overview-visual-basic"></a>列挙型の概要 (Visual Basic)
列挙体は、関連する定数のセットを操作したり、定数値を名前に関連付けたりするための便利な方法を提供します。 たとえば、一連の整数型の定数を曜日に関連付けて列挙型として宣言すると、コードで整数値ではなく曜日名を使用することができます。
